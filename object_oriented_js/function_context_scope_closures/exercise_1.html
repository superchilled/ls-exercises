Exercise 1: What is This

Read the following code carefully. What do you think is logged on line 7. Try to answer the question before you run the code. var person = {
firstName: 'Rick ',
lastName: 'Sanchez',
fullName: this.firstName + ' ' + this.lastName,
};

console.log(person.fullName);

Answer

Line 7 will log undefined undefined, because `fullName` is not a method, it isn't being being executed, and so there is not an execustion context to consider here. When the object is create, `this` is the global object and `firstName` and `lastName` are `undefined` on the global object and so this is what is set as the values for `fullName`.
